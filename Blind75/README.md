# Blind 75 — MAANG Interview Prep

This folder contains solutions to the **Blind 75** LeetCode problems ([list](https://leetcode.com/problem-list/oizxjoit/)), with:

- **Brute force** and **optimized** approaches
- **Python** and **Golang** code (in each `.md` and optionally in `.py`/`.go` files)
- **Walkthroughs**, **edge cases**, **pitfalls**, **follow-ups**, and **when to use** notes

**File naming:** `ProblemName-LC-NNN-X.md` where **X** = **E** (Easy), **M** (Medium), or **H** (Hard).  
Example: `TwoSum-LC-1-E.md`, `MergeKSortedLists-LC-23-H.md`.
